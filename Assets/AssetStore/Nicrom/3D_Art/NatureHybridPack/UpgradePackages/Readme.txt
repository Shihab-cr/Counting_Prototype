If you use the URP/HDRP pipeline, you will need to upgrade a few files for the art pack to work correctly.

1. Import the package NHP_URP_Core if your project uses URP and NHP_HDRP_Core if it uses HDRP.
2. Import the package that corresponds to the Unity version and pipeline of your project.
